\begin{problem}
  \item[2.] 下列哪一種方式可以將電中性的鎵原子（\ch{Ga}）變為鎵離子（\ch{Ga^3+}）？

    \item 使鎵原子獲得 3 個電子
    \item 使鎵原子獲得 3 個質子
    \item 使鎵原子失去 3 個電子
    \item 使鎵原子失去 3 個質子
  \end{choices}
\end{problem}
