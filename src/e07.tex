\begin{problem}
  \item[7.] 已知一骰子上標有 1、2、3、4、5、6 六種點數，且每個點數出現的機會相等。投擲一次骰子並觀察其出現的點數，試問下列敘述何者正確？
  \begin{choices}
    \item 出現 3 點的機率為 $\fraction{1}{2}$
    \item 出現的點數可被 3 整除的機率為 $\fraction{1}{2}$
    \item 出現的點數大於或等於 3 的機率為 $\fraction{1}{2}$
    \item 出現的點數為奇數的機率為 $\fraction{1}{2}$
  \end{choices}
\end{problem}
