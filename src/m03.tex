\begin{problem}
  \item[3.] 一袋中裝有 $8$ 顆紅球、$7$ 顆黑球與 $6$ 顆白球。已知每顆球被抽中的機會相等，則從袋中抽取一球時，抽中黑球的機率為何？
  \begin{choices}
    \item $\fraction{2}{7}$
    \item $\fraction{1}{3}$
    \item $\fraction{8}{21}$
    \item $\fraction{3}{7}$
  \end{choices}
\end{problem}
