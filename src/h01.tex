\begin{problem}
  \item[1.] 以下是甲、乙兩人提出的看法：
  \begin{enumerate}
    \item[(甲)] $1$ 是質數
    \item[(乙)] $49$ 是質數
  \end{enumerate}
  對於兩人的看法，下列判斷何者正確？
  \begin{choices}
    \item 兩人皆正確
    \item 兩人皆錯誤
    \item 甲正確，乙錯誤
    \item 甲錯誤，乙正確
  \end{choices}
\end{problem}
